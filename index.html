Frequently Asked Questions

How do I convert binary to hex?

Enter your binary number, select "Binary" as the input base, and the hexadecimal result appears instantly. For example, 11111111 → FF.

What is Base32 / Base36?

Base32 uses digits 0-9 and letters A-V. Base36 uses 0-9 and A-Z — the most compact single-character-digit representation using alphanumeric characters.

Can I convert negative numbers?

Yes — enter a negative number in any base and all conversions update live. Bit visualization shows the magnitude bits with a sign indicator.
